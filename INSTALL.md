# Quick Installation Guide

## Install in 3 Easy Steps

### Step 1: Open Chrome Extensions
1. Open Chrome/Chromium/Brave browser
2. Type in address bar: `chrome://extensions/`
3. Press Enter

### Step 2: Enable Developer Mode
1. Look for **Developer mode** toggle in the top-right corner
2. Click to enable it (it should turn blue/green)

### Step 3: Load the Extension
1. Click **Load unpacked** button
2. Navigate to and select the `smo-worklog-extension` folder
3. Click **Select** or **Open**

## Updating Icons (Optional)

The extension comes with placeholder icons. To use better icons:

### Online Method (Easy):
1. Visit: https://www.favicon-generator.org/
2. Upload the `icons/icon.svg` file
3. Generate all sizes (16x16, 32x32, 48x48, 128x128)
4. Download and replace the PNG files in the `icons/` folder
5. Reload the extension in `chrome://extensions/`

### Manual Screenshots (Quick):
1. Open `icons/icon.svg` in browser
2. Zoom to fill window
3. Take screenshot and crop to square
4. Resize to different sizes using any image editor
5. Save as icon16.png, icon32.png, icon48.png, icon128.png

## Troubleshooting

**Extension not showing up?**
- Make sure Developer mode is ON
- Check that you selected the correct folder
- Look for any error messages in red

**Need to reload after changes?**
1. Go to `chrome://extensions/`
2. Find "SMO Worklog Tool"
